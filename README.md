
# ellipsis
text "ellipsis" example in single line and multiple line.
https://d50000.github.io/ellipsis/

## single line

``` 
white-space:  nowrap;
overflow:  hidden;
text-overflow:  ellipsis;
width: 300px;
```

## multiple lines

```
overflow:  hidden;
display:  -webkit-box;
-webkit-line-clamp:  4;
-webkit-box-orient:  vertical;
width:  550px;
```
